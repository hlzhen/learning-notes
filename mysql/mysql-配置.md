## 2. DB、DBMS、SQL之间的关系
```markdown
# BD
 - 数据库(database)，用于存储结构化的数据，在硬盘上以文件的形式存在。

# DBMS
 - 数据库管理系统(database management system)，常见的有Mysql、Oracle、DB2、SqlServer等等。

# SQL
 - 结构化查询语言，是一门标准的通用语言。

# 三者之间的关系
 - DBMS -(执行)-> SQL -(操作)-> DB
```
